# Animation V1 Staging

This folder contains first-pass 3/4 top-down animation assets for Aethelgard-Hanba.

The runtime frames are staged for visual review. They are not wired into Phaser runtime yet.

Player frames are organized by character and state:

- `idle`
- `move`
- `attack`
- `dodge`

Enemy frames are organized by enemy and state:

- `move`
- `attack`

The weapons remain independent runtime assets and should continue to render as a separate layer from the character model.
